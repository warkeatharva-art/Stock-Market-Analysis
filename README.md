# Stock Market Analysis Dashboard (Power BI)

## 📊 Project Overview
This project focuses on analyzing stock market data using Power BI to understand price movements, trading volumes, and market trends over time. The dashboard provides an interactive view of stock performance with key financial indicators.

---

## 🎯 Objectives
- Analyze stock price trends over time  
- Track trading volume and market activity  
- Identify patterns in stock performance  
- Provide insights for better financial understanding  

---

## 📁 Dataset
- Stock market dataset (NSE listed companies)  
- Includes:
  - Open, High, Low, Close prices  
  - Trading volume  
  - Date-wise stock data  

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Excel (Data Cleaning & Preparation)  

---

## 📈 Dashboard Features
- KPI cards (Open, High, Low, Close prices)  
- Time filters (1D, 5D, 1M, 6M, YTD, 1Y, etc.)  
- Price trend analysis using line charts  
- Volume analysis with bar charts  
- Quarterly and yearly comparison  
- Interactive slicers for dynamic analysis  

---

## 🔍 Key Insights
- Identified upward price trend in recent months  
- Observed fluctuations in trading volume across different periods  
- Detected key resistance and support levels in stock prices  
- Analyzed seasonal and time-based variations in stock performance  

---

## 📂 Repository Contents
- `stock-market-dashboard.pbix` → Power BI file  
- `dataset.xlsx` → Dataset used  
- `images/` → Dashboard screenshots  

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Use filters to explore stock trends  

---

## 📌 Conclusion
This project demonstrates how Power BI can be used to analyze financial data and visualize stock market trends for better insights and decision-making.
